# QuickPOS Landing Page

A professional, responsive, and pixel-perfect single-page landing website for the QuickPOS Point of Sale system, built using PHP.

## Project Overview
This project focuses on delivering a modern, clean, and professional UI/UX. It includes sections for Features, Pricing, and a functional Contact Us form that uses PHP to validate inputs and simulate a success response.

## Prerequisites
To run this project locally, you will need:
**PHP** (v7.4 or higher recommended) installed on your machine.
Alternatively, you can use a local server environment like **XAMPP**, **WAMP**, or **MAMP**.
**Git** to clone the repository.

## Project Setup Instructions

Follow these steps to get the project running on your local machine:

**1. Clone the repository**
Open your terminal or command prompt and run:
`git clone [Insert your GitHub repository URL here]`

**2. Navigate to the project directory**
`cd [Insert your repository folder name here]`

**3. Start the local PHP server**
If you have PHP installed globally, you can use PHP's built-in development server. Run the following command in the root of the project directory:
`php -S localhost:8000`

*(Note: If you are using XAMPP/WAMP/MAMP, move the project folder into your `htdocs` or `www` directory and start your Apache server from the control panel).*

**4. View the Landing Page**
Open your preferred web browser and navigate to:
`http://localhost:8000` 
*(Or `http://localhost/your-folder-name` if using XAMPP/WAMP).*
